# LargeFileUploads
Vencord plugin for automatically detecting, and uploading files that exceed max upload limit to a hosting service and sending the link instead.

## Supports:
**Catbox:**
- 200MB max size
- No expiry
- Embed available

**Litterbox:**
- 1GB max size
- 3 days expiry time
- Embed available

**Uguu:**
- 128MB max size
- 3 hours expiry time
- Embed available

**Custom Zipline instance:**
- Custom
- Custom
- Embed available
